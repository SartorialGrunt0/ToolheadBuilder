Welcome to Toolhead Builder — a structured catalog of modern 3D‑printing toolheads and the components that make them work — **extruders**, **hotends**, and **probes**. 

Each section is seperated into three pages:
- Overview
- Community Picks
- Catolog

This aims to help new builders learn about the different components and get familiar with
the terminology while also quickly narrowing down the community recommendations.

In addition, hopefully this helps make it easier to find community projects and discover something new.
