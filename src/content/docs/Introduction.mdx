Welcome to Toolhead Builder — a structured catalog of modern 3D‑printing toolheads and the components that make them work — **extruders**, **hotends**, and **probes**.
This page is a continuous work in progress as information is filled in and new components are discovered.

The toolhead configurator page has a list of some of the popular community toolheads to pick from.
Once a toolhead is selected, the list of compatible parts is shown.
These lists are compiled from the toolheads’ confirmed compatible parts.
In addition, any hardware that should be compatible can be shown based on the parts’ metadata.

This page also offers some standalone component sections to browse through.
This aims to help new builders learn about the different components and get familiar with
the terminology while also narrowing down the to the community recommendations.

Each componet section is seperated into three pages:

- Overview
- Community Picks
- Catolog

Hopefully this page helps you find community projects and discover something new. Please feel
free to open up an issue on the github page to provide suggestions, request new componet additons, and report bugs.
